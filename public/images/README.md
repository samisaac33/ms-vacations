# Imágenes de propiedades

Coloca aquí las fotos reales de cada alojamiento. La estructura esperada es:

```
public/images/
  destinations/
    san-clemente.jpg
    portoviejo.jpg
  properties/
    alojamiento-arrecife/
      1.jpg
      2.jpg
      ...
```

Los nombres y rutas están definidos en `data/properties.json`. Reemplaza los archivos actuales por tus fotos reales sin cambiar los nombres, o actualiza las rutas en ese archivo.
